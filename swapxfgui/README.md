## What is 袇楔饝珱饝珤鉂奨FG?

袇楔饝珱饝珤鉂奨FG is a secure wallet and non-custodial decentralized exchange rolled into one application. Store your coins,
trade peer-to-peer with minimal fees and never give up control over your digital assets.

袇楔饝珱饝珤鉂奨FG has been tested on the following platforms:

- Windows 10
- Linux (Ubuntu 18.04+)
- macOS (10.14 - 11.0)


## Get Started
